# Redis 小助手 UI 设计文档

## 界面布局

### 整体结构

```
┌─────────────────────────────────────────────────────────────┐
│ 菜单栏（蓝色背景）                                            │
│ [连接 ▼][DB ▼][更多 ▼][多选]                                 │
├─────────────────────────────────────────────────────────────┤
│ 状态栏：当前服务器 > DB > key 信息                             │
├───────────────────────────┬─────────────────────────────────┤
│ 键列表区（35%宽度）          │ 值展示区（65%宽度）               │
│ ┌───────────────────────┐ │                                 │
│ │ [🔄] [搜索框] [搜索] [+]│ │    类型: STRING                │
│ └───────────────────────┘ │    ┌─────────────────────┐     │
│ ┌───────────────────────┐ │    │ 值编辑器            │     │
│ │                       │ │    │                     │     │
│ │   键列表 (el-tree)     │ │    │                     │     │
│ │                       │ │    └─────────────────────┘     │
│ │                       │ │    [删除] [修改]               │
│ └───────────────────────┘ │                                │
│ ┌───────────────────────┐ │                                │
│ │ 全选 ▼  取消  [移入(N)]│ │                                │
│ │ 快捷键提示             │ │                                │
│ ├───────────────────────┤ │                                │
│ │ [加载更多] N/M [加载所有]│ │                                │
│ └───────────────────────┘ │                                │
└───────────────────────────┴─────────────────────────────────┘
```

## 菜单栏

### 位置
顶部固定，蓝色背景（#1890ff）

### 元素

| 元素 | 类型 | 说明 |
|------|------|------|
| 连接 | 下拉菜单 | 选择/管理 Redis 服务器 |
| DB | 下拉菜单 | 选择数据库、废键箱、新增/删除 DB |
| 更多 | 下拉菜单 | 导入、导出、清空 |
| 多选 | 按钮 | 切换多选模式 |

### 样式
- 紧凑排列，无间隙（gap: 0）
- 菜单项：透明背景，白色文字，无圆角
- Hover：半透明白色背景
- 多选按钮激活状态：橙色背景

## 搜索栏

### 位置
键列表区顶部

### 元素

| 元素 | 类型 | 说明 |
|------|------|------|
| 🔄 | 按钮 | 刷新键列表 |
| 搜索框 | 输入框 | 支持 * 通配符搜索 |
| 搜索 | 按钮 | 执行搜索 |
| + | 按钮 | 添加新键（绿色） |

### 样式
- 按钮统一高度 36px
- + 按钮：绿色（success），hover 变浅绿

## 键列表

### 多选模式

#### 触发方式
点击菜单栏"多选"按钮

#### 展开面板
在键列表上方展开（向上）：

```
┌─────────────────────────────────────────┐
│ 全选 ▼  取消  [移入 (N)]                 │
│ 快捷键: Ctrl+A | Ctrl+Shift+A | Esc     │
├─────────────────────────────────────────┤
│ ☐ key1                                  │
│ ☐ key2                                  │
└─────────────────────────────────────────┘
```

#### 操作按钮

| 按钮 | 功能 |
|------|------|
| 全选 ▼ | 下拉选择：全选当前已加载 / 全选所有 |
| 取消 | 清空所有选择 |
| 移入 (N) | 将选中的 N 个键移入废键箱 |

#### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+A | 全选当前已加载 |
| Ctrl+Shift+A | 全选所有 |
| Esc | 退出多选模式 |

## 分页区域

### 位置
键列表底部

### 元素

| 元素 | 类型 | 说明 |
|------|------|------|
| 加载更多 | 按钮 | 加载下一批 keys |
| N/M | 文本 | 已加载数量 / 总数 |
| 加载所有 | 按钮 | 加载全部 keys |

### 样式
- 按钮统一高度，透明背景
- 居中对齐，间距 8px
- 左右留 12px 空隙

## 值展示区

### 布局
- 顶部：类型标签
- 中部：值编辑器（根据类型不同）
- 底部：删除、修改按钮

### 类型支持
- STRING：文本编辑器
- HASH：键值对表格
- LIST：列表项
- SET：集合项
- ZSET：有序集合

## 颜色规范

### 主色调
- 主色：#1890ff（蓝色）
- 成功：#67c23a（绿色）
- 警告：#e6a23c（橙色）
- 危险：#f56c6c（红色）

### 背景色
- 菜单栏：#1890ff
- 多选面板：#fffbeb（淡黄色）
- 按钮默认：透明

## 交互规范

### 按钮
- 默认：透明底、灰边框
- Hover：蓝底白字（主色）或对应色
- 激活：对应主题色

### 下拉菜单
- 触发：点击
- 位置：下方展开
- 动画：淡入淡出

### 折叠面板
- 多选面板：向上展开
- 过渡：平滑动画
