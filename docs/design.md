## 1. 项目概述

### 1.1 项目背景

Online Redis Manager 是一个符合中国开发者思维方式的在线 Redis 管理工具，设计初衷是快捷、简单、实用。该工具采用前后端分离架构，支持 Redis 服务器的管理和数据操作。

### 1.2 核心功能

- **多服务器管理**：支持添加、编辑、删除 Redis 服务器配置
- **数据库管理**：支持切换和管理不同的 Redis 数据库
- **键值操作**：支持查看、添加、编辑、删除 Redis 键值
- **数据类型支持**：支持 String、List、Set、ZSet、Hash 五种核心数据类型
- **搜索功能**：支持按模式搜索键
- **Excel 导入导出**：支持批量操作数据
- **配置存储**：支持本地存储和服务器存储两种模式

### 1.3 运行环境

- **前端**：浏览器
- **后端**：PHP 5.6+、PhpRedis 扩展
- **Web 服务器**：Apache、Nginx 或 PHP 内置服务器
- **Redis**：2.8+ 版本

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 服务器唯一标识 |

### 5.2 Redis 数据类型

| 类型 | 描述 | 操作 |
|------|------|------|
| String | 字符串 | set, get, incr, decr |
| List | 列表 | lpush, rpush, lrange |
| Set | 集合 | sadd, smembers, srem |
| ZSet | 有序集合 | zadd, zrange, zrem |
| Hash | 哈希 | hset, hget, hgetall |

## 8. 构建与部署

### 8.1 环境要求

- **PHP**：5.6+
- **PhpRedis**：2.2.8+
- **Web 服务器**：Apache、Nginx 或 PHP 内置服务器
- **Redis**：2.8+

### 8.2 安装步骤

1. **安装 Web 服务器**：如 Wamp、Lamp、Mamp 等
2. **安装 PhpRedis 扩展**：参考 [安装说明](install/README.md)
3. **部署代码**：将代码部署到 Web 服务器
4. **设置权限**：设置 log 和 upload 目录权限为 0777
5. **访问应用**：浏览器打开 http://localhost/www/index.html

### 8.3 配置说明

- **服务器配置模式**：在 www/js/main.js 中配置 isConfigLocal
  - isConfigLocal: true：服务器配置存储在本地
  - isConfigLocal: false：服务器配置存储在服务器
