# Redis 小助手 - UI设计文档

## 项目概述

这是一个**Redis可视化管理工具**，提供可视化的Redis服务器管理、数据库操作、键值对查看与编辑、多选删除、废键箱等功能。采用 Tauri + Vue3 技术栈，支持多服务器连接管理。

---

## 二、标签栏设计

### 2.1 位置与样式

- **位置**：菜单栏下方、状态栏上方
- **显示条件**：当存在两个及以上标签时显示，可通过菜单切换
- **样式**：Element Plus Tabs 组件，与整体风格一致

### 2.2 标签项

| 元素 | 说明 |
|------|------|
| 标签标题 | 显示"服务器名 > DB编号"格式 |
| 关闭按钮 | 鼠标悬停时显示，点击关闭标签 |
| 新建按钮 | 标签栏最右侧的"+"按钮 |

### 2.3 交互

| 操作 | 行为 |
|------|------|

| 点击"+" | 弹出服务器选择对话框，创建新标签 |

## 五、键列表区设计

### 5.1 正常模式

- 使用 `el-tree` 展示键列表
- 点击选中，高亮显示
- 支持滚动加载

### 5.2 多选模式

#### 触发方式
点击菜单栏"多选"按钮

#### 展开面板（向上展开）

```
┌─────────────────────────────────────────┐
│ 全选 ▼  取消  [删除 (N)]                 │  ← 淡黄色背景 #fffbeb
│ 快捷键: Ctrl+A | Ctrl+Shift+A | Esc     │
├─────────────────────────────────────────┤
│ ☐ key1                                  │
│ ☐ key2                                  │
│ ...                                     │
└─────────────────────────────────────────┘
```

#### 操作按钮

| 按钮 | 功能 |
|------|------|
| 全选 ▼ | 下拉选择：全选当前已加载 / 全选所有 |
| 取消 | 清空所有选择 |
| 删除 (N) | 将选中的 N 个键删除废键箱 |

#### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+A | 全选当前已加载 |
| Ctrl+Shift+A | 全选所有（先加载全部） |
| Esc | 退出多选模式 |

#### 技术实现

- 单 `el-tree` 实例，动态绑定 `:show-checkbox="isMultiSelectMode"`
- 选中状态通过 `treeRef.setCheckedKeys()` 控制
- 面板使用 `el-collapse-transition` 实现平滑展开/收起

## 九、日志对话框设计（慢日志 + 操作审计）

### 9.1 入口

菜单栏 **「更多 ▼」** → **「慢日志」**

### 9.2 布局

使用 Element Plus Dialog 对话框：

| 区域 | 内容 |
|------|------|
| 搜索栏 | 关键词搜索 + 过滤选项 |
| 日志列表 | 滚动列表，展示慢日志条目 |
| 日志详情 | 每条日志显示：命令内容、执行时间、客户端IP、时间戳 |

### 9.3 交互

| 操作 | 行为 |
|------|------|
| 搜索 | 按关键词过滤日志 |
| 自动滚动 | 新日志自动滚动到底部 |
| 关闭对话框 | 停止获取日志 |

### 9.4 操作审计视图

日志对话框通过顶部 Tab 切换"慢日志"和"操作审计"两个视图。

#### 筛选栏

| 元素 | 类型 | 说明 |
|------|------|------|
| 关键词搜索 | 输入框 | 搜索命令/参数 |
| 命令类型 | 下拉选择 | GET/SET/DEL/HSET/HGET 等 13 种命令 |
| 时间范围 | 日期时间范围选择器 | 按时间段筛选 |
| 搜索按钮 | 主色按钮 | 执行筛选 |
| 清空日志 | 按钮 | 清空所有审计日志 |
| 生成测试数据 | 按钮 | 生成 100 条测试数据 |

#### 审计日志表格

| 列 | 说明 |
|------|------|
| 时间 | 操作时间戳 |
| 命令 | Redis 命令名（GET/SET/DEL） |
| 参数 | 命令参数 |
| 耗时 | 执行耗时（ms） |
| 状态 | 成功/失败 |
| 错误 | 错误信息（如有） |

#### 统计卡片

| 卡片 | 说明 |
|------|------|
| 总命令数 | 所有命令调用次数之和 |
| 平均响应时间 | 加权平均耗时（ms） |
| 成功率 | 成功命令占比（%） |
| 最慢命令 | 平均耗时最高的命令 |

#### 命令分布图

- 使用纯 CSS 水平条形图展示各命令调用次数
- 按调用次数降序排列
- 每种命令有固定颜色映射

#### 分页

- 默认每页 50 条
- 支持上一页/页码/下一页导航

## 十四、操作流程

### 14.1 多选删除流程

```
1. 点击菜单栏【多选】按钮
   ↓
2. 键列表显示复选框，上方展开操作面板
   ↓
3. 勾选需要删除的键（或 Ctrl+A 全选）
   ↓
4. 点击【删除(N)】按钮
   ↓
5. 确认对话框
   ↓
6. 键被删除废键箱，列表刷新
```

### 14.2 废键箱恢复流程

```
1. 点击【DB ▼】→【废键箱】
   ↓
2. 查看已删除的键列表
   ↓
3. 勾选需要恢复的键
   ↓
4. 点击【恢复】按钮
   ↓
5. 键恢复到原位置
```
